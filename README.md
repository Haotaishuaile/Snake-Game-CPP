# 🐍 贪吃蛇进阶版 (Advanced Snake Game)

一款基于 C++ 和 EasyX 图形库开发的经典贪吃蛇游戏。在保留经典贪吃蛇核心玩法的基础上，引入了多级难度、丰富的音乐播放系统，以及极具策略性的“海克斯强化”玩法。

## ✨ 核心特性 (Features)

* 🎮 **多维游戏模式**：包含入门、进阶、高级三大基础模式，难度与机制递进。
* 🎲 **独创“海克斯”模式**：引入生命值系统与定时强化机制（如“死亡而已”、“金钱至上”、“极致燃烧”等 Buff），极大地丰富了游戏的策略性与可玩性。
* 🎵 **内置音乐播放器**：集成底层 `mciSendString` 音频控制，支持在主菜单自由切换多首背景音乐（青花瓷、卜卦等）。
* ⚡ **极致丝滑的操控**：摒弃传统的控制台输入拦截，采用 Windows 底层 `GetAsyncKeyState` 异步按键检测。完美兼容大小写输入与方向键，彻底解决输入法冲突造成的按键失灵问题。
* 🚀 **高性能渲染架构**：采用内存预加载图片资源结合 EasyX 双缓冲绘图技术，消除了高频磁盘 I/O 带来的画面卡顿与闪烁，保证游戏高帧率流畅运行。
* 💾 **本地数据持久化**：支持多用户账号登录，通过文件流操作安全读写历史最高分记录。

## 🛠️ 开发环境与依赖 (Environment)

* **开发工具**：Visual Studio 2022 (或支持 C++ 的其他 VS 版本)
* **核心语言**：C++
* **图形引擎**：[EasyX 图形库](https://easyx.cn/)（编译前需确保已将 EasyX 安装至对应的 VS 环境中）
* **系统支持**：Windows (依赖 `Winmm.lib` 静态库支持音频播放)

## 🚀 安装与运行指南 (How to Run)

1.  克隆本仓库到本地，或下载 ZIP 压缩包并解压。
2.  前往 EasyX 官网下载并安装最新版图形库。
3.  进入项目根目录，双击打开 `贪吃蛇.sln` 解决方案文件。
4.  在 Visual Studio 顶部工具栏中，将编译模式设置为 `Debug` 或 `Release`，架构选择 `x64` 或 `x86`。
5.  按下 `F5` 或点击“本地 Windows 调试器”即可编译并运行游戏。

## 🕹️ 操作说明 (Controls)

* **移动**：`W` `A` `S` `D` 或 键盘方向键 `↑` `↓` `←` `→`
* **退出游戏/保存记录**：`P` 键
* **菜单选择**：鼠标左键点击界面对应按钮
